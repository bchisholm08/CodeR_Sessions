What is a CodeR Session?

The CodeR-Sessions is an open-sourced project designed to help people practice statistics and coding in R. Here's what we do:

The CodeR Sessions is a project focused on highlighting the open source resources available for individuals wanting to learn R. Data science, coding, and statistics are all fields becoming more prominent. Statistics plays a large role in these realms, and R provides a route for teaching yourself skills that are truly applicable to your job. The website scrapes the web for viable datasets based on a set of parameters from algorithm code in THIS GitHub repo. After a suitable dataset is found, a set of research questions (RQ's) is generated that relate to the data. Once one of these combinations of a data set and RQ's is found, it's saved as a query— a prompt in the form of a PDF document that is ready for the user to practice R with!

The website scrapes the web for datasets and automatically generates research questions (RQ) related to the data. Once a combination of data and RQs is found, it's saved as a query, a question prompt that includes both the dataset and RQs ready for you to practice with!

For example, a query could look like this:

Analyzing NFL Rushing Performance

pqn:45968
Player ID  Yards  Play Number  Run Direction  Defense Formation  Result Type
player1    45     12           left           4-3                 First Down
player2    30     8            right          Nickel              No Gain
player3    50     15           center         3-4                 Touchdown
player4    15     20           left           Dime                First Down
player5    5      25           right          4-3                 No Gain

Research Questions:

  • Explore the relationship between temperature and yards gained. What temperature results in the most yards?
  • Can we predict the expected yards gained (yards) based on the running direction (runDir) and the number of plays (playNum)? What statistical model best fits the data?
  • Is there a correlation between the running direction (runDir) and average yards gained?

Note: Data above is simulated.

Each query is assigned a unique Problem Query Number (PQN) for quick access. After completing your analysis, you can post your solution and share it on the problem page for others to compare or ask for help!

Additionally, users can upload their own datasets and create custom queries, challenging others to solve new problems. These user-generated queries will also be saved to the database, ready to be tried by anyone!
